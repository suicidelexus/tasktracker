# Руководство по новым возможностям Task Tracker

## Справка

### Как открыть справку:
1. Нажмите на иконку "❓" (знак вопроса) в нижней части бокового меню
2. Откроется модальное окно со справочной информацией
3. Используйте боковое меню слева для навигации между разделами
4. Закрыть можно кликом вне окна или по кнопке ✕

### Разделы справки:
- **Priority Score** - полное описание методики оценки задач

## Фильтрация данных (Excel-style)

### Где доступна:
- ✅ Priority Score (все столбцы)
- ✅ Все задачи (все столбцы)
- ✅ Проекты (все столбцы)
- ✅ Завершенные задачи (все столбцы)

### Как использовать:

1. **Нажмите на иконку воронки** рядом с названием столбца
2. **Для текстового поиска** (столбец "Название"):
   - Введите ключевые слова в поле поиска
   - Фильтрация происходит по названию и описанию задачи

3. **Для выбора значений**:
   - Отметьте нужные значения галочками
   - Используйте "Выбрать все" для массового выбора
   - "Очистить" для сброса фильтра по столбцу

4. **Сброс всех фильтров**:
   - Нажмите кнопку "Сбросить фильтры" в шапке страницы

### Индикация активных фильтров:
- Иконка воронки становится **синей** если фильтр активен
- Иконка воронки **серая** если фильтр не применен

## Priority Score

### Что изменилось:
- Раздел показывает **только черновики** для предварительной оценки
- Описание методики перенесено в Справку
- Задачи отсортированы по Score (от большего к меньшему)

### Как работать:

1. **Создать черновик**:
   - Нажмите "Создать черновик задачи"
   - Заполните название и метрики Priority Score
   - Сохраните

2. **Перенести во все задачи**:
   - Нажмите кнопку "Перенести" справа от задачи
   - Дополните информацию (исполнитель, проект и т.д.)
   - После переноса задача исчезнет из Priority Score

3. **Фильтрация**:
   - Используйте иконки воронки в заголовках столбцов
   - Можно фильтровать одновременно по нескольким параметрам

### Интерпретация Score:
- **40+** → Берём в работу ✅
- **25-39** → Кандидат в работу (проверка)
- **10-24** → Требуется уточнение ⚠️
- **0-9** → Не берём в работу / Icebox ❌

## Матрица Эйзенхауэра

### Drag & Drop:

1. **Перетащить задачу**:
   - Наведите курсор на задачу (курсор изменится)
   - Зажмите левую кнопку мыши
   - Перетащите в другой квадрант
   - Отпустите кнопку мыши
   - Изменения автоматически сохранятся

2. **Квадранты**:
   - 🔴 Важно и срочно → Делать немедленно
   - 🔵 Важно, не срочно → Планировать и делать
   - 🟡 Не важно, срочно → Делегировать
   - ⚪ Не важно, не срочно → Удалить или отложить

### Просмотр всех задач квадранта:

1. **В компактном виде** показывается первые 7 задач
2. Если задач больше:
   - Появляется ссылка "+ еще N задач(и)"
   - Или иконка развернуть в заголовке квадранта
3. **Клик по ссылке/иконке** → открывается модальное окно
4. В модальном окне отображаются **все задачи** квадранта
5. Drag & Drop работает и в модальном окне

### Визуальные изменения:
- Карточки задач стали **компактнее**
- Показывается только **название задачи**
- Убран исполнитель и группа из карточки
- Hover эффекты при наведении

## Горячие клавиши (будущее)

Планируется добавить:
- `Ctrl + K` - открыть поиск
- `Ctrl + /` - открыть справку
- `Esc` - закрыть модальные окна
- `N` - создать новую задачу

## Советы по работе

### Рабочий процесс:

1. **Сбор идей** → Priority Score (черновики)
2. **Оценка** → рассчитываем Score
3. **Приоритизация** → переносим в работу (Score 40+)
4. **Планирование** → Матрица Эйзенхауэра
5. **Выполнение** → Все задачи
6. **Завершение** → Завершенные

### Фильтрация задач:

- Используйте **множественные фильтры** для точного поиска
- Фильтры работают по логике **AND** (все условия)
- Не забывайте **сбрасывать фильтры** после работы

### Drag & Drop:

- Перетаскивайте задачи для **быстрой приоритизации**
- Изменения сохраняются **автоматически**
- Если ошибка - просто перетащите обратно

## Управление проектами

### Создание проекта:
1. Нажмите **"+"** рядом с "Все задачи" в боковом меню
2. Заполните название, описание
3. Выберите иконку из доступных эмодзи
4. Сохраните

### Редактирование проекта:
1. Откройте проект из бокового меню
2. Нажмите на **иконку карандаша** ✏️ рядом с названием
3. Измените данные
4. Сохраните

### Удаление проекта:
1. Откройте проект из бокового меню
2. Нажмите на **иконку корзины** 🗑️ рядом с названием
3. Подтвердите удаление
4. **Важно:** Задачи проекта будут перенесены в "Без проекта", а не удалены

### Скрытие/показ проектов:
- Нажмите **стрелку** рядом с "Все задачи" для сворачивания списка проектов

## Импорт/Экспорт

### Загрузка задач из Excel:
1. Скачайте шаблон через меню "Новая задача" → "Скачать шаблон Excel"
2. Заполните файл (не все поля обязательны)
3. Загрузите через "Новая задача" → "Загрузить из Excel"
4. При ошибках импорт остановится с указанием проблемы

### Экспорт задач:
Планируется добавить кнопку "Экспорт в Excel" для выгрузки задач
