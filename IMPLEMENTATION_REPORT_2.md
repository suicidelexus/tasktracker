## 🎯 Архитектура изменений:

### Drag & Drop в матрице:
```javascript
// Обработчики
handleDragStart → устанавливает draggedTask, меняет opacity
handleDragEnd → сбрасывает opacity
handleDragOver → предотвращает default, устанавливает dropEffect
handleDragEnter → устанавливает isDragOver = true
handleDragLeave → устанавливает isDragOver = false
handleDrop → обновляет задачу на backend, обновляет matrix

// Визуальные эффекты
- opacity: 0.5 при перетаскивании
- синяя рамка на квадранте при наведении
- курсор: grab/grabbing
```

### Excel-фильтры в AllTasks:
```javascript
// Структура фильтров
{
  search: '',         // строка для поиска
  assignee: [],       // массив выбранных исполнителей
  priority: [],       // массив выбранных приоритетов
  project_id: [],     // массив ID выбранных проектов
  rice_category: []   // массив выбранных категорий Score
}

// Применение фильтров
loadData() → клиентская фильтрация по всем массивам
```

### Управление проектами:
```javascript
// Создание
Sidebar → "+" → ProjectModal → projectsAPI.create()

// Редактирование
ProjectView → Edit2 icon → ProjectModal(project) → projectsAPI.update()

// Удаление
ProjectView → Trash2 icon → confirm → projectsAPI.delete()
Backend: перенос задач в NULL → удаление проекта
```
