✅ Полностью рабочий REST API

   - Страница Rice Scoring
   - Страница Матрицы Эйзенхауэра
✅ React Router для навигации
✅ Axios для работы с API
✅ Lucide Icons для иконок

### 3. Функциональность
✅ **Управление задачами:**
   - Создание, редактирование, удаление
   - Название, описание, ссылка на внешний трекер
   - Назначение исполнителя
   - Привязка к группе проекта

✅ **Группировка:**
   - Создание групп для разных проектов
   - Фильтрация задач по группам

✅ **Rice Scoring:**
   - Value (1-5) — влияние
   - Reach (1-5) — охват
   - Budget Impact (1, 1.2, 1.5, 2) — влияние на бюджет
   - Confidence (10-100%) — уверенность
   - Автоматический расчет: Priority Score = Value × Reach × Budget × Confidence
   - Категоризация: "В работу" (40+), "Кандидат" (25-39), "Требуется уточнение" (10-24), "Не берём" (0-9)

✅ **Матрица Эйзенхауэра:**
   - 4 квадранта: Важно/Срочно
   - Визуальная сетка с задачами
   - Автоматическая классификация

## 🚀 КАК ЗАПУСТИТЬ

### Самый простой способ:
**Двойной клик на `START_ALL.bat`** — запустит backend и frontend автоматически!

### Альтернативные способы:

#### Вариант 1: Отдельные bat-файлы
1. Запустить `start_backend.bat`
2. Запустить `start_frontend.bat`

#### Вариант 2: Через командную строку
```powershell
# Terminal 1 - Backend
cd C:\Users\suici\PycharmProjects\tasktracker
py main.py

# Terminal 2 - Frontend
cd C:\Users\suici\PycharmProjects\tasktracker\frontend
npm start
```
