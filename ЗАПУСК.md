# Запуск Task Tracker

## Быстрый запуск

1. Откройте проводник Windows
2. Перейдите в папку проекта: `C:\Users\suici\PycharmProjects\tasktracker`
3. **Дважды кликните** на файл `START_ALL.bat`
4. Откроются 2 окна:
   - **Зелёное окно** - Backend Server (порт 8080)
   - **Чёрное окно** - Frontend Server (порт 3000)
5. Подождите ~30 секунд пока серверы запустятся
6. Браузер автоматически откроется на http://localhost:3000

## Альтернативный запуск (вручную)

### Backend
```bash
cd C:\Users\suici\PycharmProjects\tasktracker
py -m uvicorn main:app --host 0.0.0.0 --port 8080 --reload
```

### Frontend
```bash
cd C:\Users\suici\PycharmProjects\tasktracker\frontend
npm start
```

## Проверка работы

- Backend API: http://localhost:8080/api/health
- Frontend: http://localhost:3000
- API документация: http://localhost:8080/docs

## Доступ из локальной сети

1. Узнайте свой IP адрес:
   ```bash
   ipconfig
   ```
2. Найдите "IPv4 Address" (например, 192.168.1.100)
3. Другие пользователи в вашей сети могут открыть:
   - http://ВАШ_IP:3000 (Frontend)
   - http://ВАШ_IP:8080 (Backend API)

## Остановка серверов

Просто закройте окна Backend и Frontend.
