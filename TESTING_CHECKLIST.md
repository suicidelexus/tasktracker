## 1. ✅ Drag & Drop в Матрице Эйзенхауэра

**Где:** http://localhost:3000/analytics/eisenhower

### Тесты:
- [ ] Наведите курсор на задачу → курсор меняется на "grab"
- [ ] Зажмите задачу → задача становится полупрозрачной (opacity 0.5)
- [ ] Перетащите на другой квадрант → квадрант подсвечивается синей рамкой
- [ ] Отпустите задачу → задача перемещается, сохраняется на backend
- [ ] Обновите страницу → задача осталась в новом квадранте
- [ ] Попробуйте перетащить в модальном окне (если задач >7)

**Ожидаемый результат:** Задачи свободно перетаскиваются, изменения сохраняются

## 2. ✅ Excel-фильтры во всех таблицах

### Где тестировать:
- [ ] "Все задачи" - http://localhost:3000/
- [ ] "Проект" - http://localhost:3000/project/1
- [ ] "Завершенные" - http://localhost:3000/completed
- [ ] "Priority Score" - http://localhost:3000/analytics/rice

### Тесты фильтров:
**Фильтр по названию (поиск):**
- [ ] Нажмите на воронку в столбце "Название"
- [ ] Введите часть названия задачи
- [ ] Проверьте что отображаются только подходящие задачи
- [ ] Иконка воронки стала синей

**Фильтр по исполнителю:**
- [ ] Нажмите на воронку в столбце "Исполнитель"
- [ ] Выберите несколько исполнителей галочками
- [ ] Нажмите "Выбрать все" → все исполнители выбраны
- [ ] Нажмите "Очистить" → все сбросилось

**Фильтр по приоритету:**
- [ ] Выберите "High" и "Highest"
- [ ] Проверьте что показываются только эти приоритеты
- [ ] Иконка воронки синяя

**Фильтр по проекту (только в "Все задачи"):**
- [ ] Выберите один или несколько проектов
- [ ] Проверьте фильтрацию

**Фильтр по Priority Score:**
- [ ] Выберите "В работу"
- [ ] Проверьте что показываются только задачи с Score 40+

**Множественные фильтры:**
- [ ] Примените фильтры по 2-3 столбцам одновременно
- [ ] Проверьте что фильтры работают по логике AND (все условия)

**Сброс фильтров:**
- [ ] Нажмите кнопку ✕ в столбце "Действия"
- [ ] Все фильтры сброшены, иконки стали серыми

**Ожидаемый результат:** Фильтры работают быстро, без лагов, правильно фильтруют данные
